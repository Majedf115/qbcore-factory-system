### Features

#### Eye Tracking System
- No green arrows or ground markers
- Clean, immersive interactions
- Support for both ox_target and qb-target
- Configurable interaction distances
- Custom icons for each interaction

#### Bucket Routing
- Prevents player overlapping
- Automatic instance creation
- Max players per instance
- Unique bucket IDs
- Server-side validation

#### Props System
- Laptop on desk (opens UI terminal)
- Water barrel (fill sprayer interaction)
- Workbench (package product interaction)
- Customizable prop positions
- Easy prop model swapping

#### Configuration
- Target system selection
- Interior ID customization
- Bucket settings
- Prop customization
- Notification messages
- Debug mode toggle
